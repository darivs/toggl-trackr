# CLAUDE.md

## Overview

Toggl Trackr is a time tracking dashboard application with a Vite-powered frontend and Express backend.

## Tech Stack

- **TypeScript** — type-safe code throughout
- **Vite** — frontend build tooling and dev server
- **Express** — backend API server
- **Drizzle ORM** — type-safe database queries
- **PostgreSQL** — relational database

## Coding Agents

**Botcha** (the AI project manager) delegates implementation tasks to **Codex CLI** (you). Botcha handles planning and task breakdown; Codex CLI handles code execution.
